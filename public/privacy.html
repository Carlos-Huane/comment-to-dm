Quién es responsable

Esta aplicación (comment-to-dm) es un proyecto personal desarrollado y operado por Carlos Jesús Huane Sarmiento, persona natural residente en Perú. No forma parte de ninguna empresa registrada.

Es un bot serverless que responde automáticamente con un mensaje directo (DM) cuando alguien comenta una palabra clave específica en publicaciones de la cuenta de Instagram del propio operador (@carloshuane). El código fuente completo está disponible como software libre en GitHub bajo licencia MIT.

Qué datos se procesan

Cuando alguien comenta en una publicación de la cuenta suscrita, Meta envía un evento webhook a este servicio con los siguientes datos públicos del comentario:

  • Nombre de usuario público del comentador (username)
  • ID de usuario de Instagram del comentador
  • Texto del comentario
  • ID del comentario
  • ID de la publicación (media_id)
  • Timestamp del evento

No se solicitan, almacenan ni procesan datos privados como correos electrónicos, números de teléfono, ubicación, información de contactos, ni ningún dato personal fuera del comentario público.

Cómo se usan los datos

Los datos recibidos se usan exclusivamente para una única acción automatizada:

  1. Verificar si el texto del comentario contiene una palabra clave predefinida (por ejemplo, "link").
  2. Si la palabra clave está presente, enviar un único mensaje directo (Private Reply) al comentador con una URL relacionada al contenido de la publicación.

El DM enviado es una respuesta única al comentario específico (mecanismo Private Replies de Meta). No se envían mensajes masivos, no se agrega al usuario a listas de contactos, ni se le contacta después.

Almacenamiento y retención

Sin base de datos Este servicio es completamente stateless. Ninguna información personal se guarda en disco, base de datos, ni ninguna forma de almacenamiento persistente.

Cada evento se procesa en memoria durante unos milisegundos y se descarta al finalizar. Los únicos registros temporales son los logs operativos de la plataforma de hosting (Vercel), que se retienen por hasta 30 días para fines de diagnóstico técnico y se eliminan automáticamente después.

Terceros involucrados

Este servicio interactúa con las siguientes plataformas de terceros para funcionar:

  • Meta Platforms, Inc. — proveedor de la Instagram Graph API. Su política: privacycenter.instagram.com/policy
  • Vercel Inc. — plataforma de hosting donde corre el servicio. Su política: vercel.com/legal/privacy-policy

Tus derechos como usuario

Como usuario de Instagram que interactúa con publicaciones donde este bot está activo, tienes derecho a:

  • No comentar la palabra clave (evita cualquier interacción)
  • Eliminar tu comentario en cualquier momento desde Instagram
  • Bloquear la cuenta operadora del bot
  • Reportar el bot a Meta si consideras que abusa de la plataforma

Eliminación de datos

Data Deletion Instructions Este servicio no almacena datos personales, por lo que no existe una base de datos que eliminar. Cada interacción es procesada en tiempo real y descartada inmediatamente.

Aún así, si querés dejar de interactuar con el servicio:

  1. Abrí Instagram → tu perfil → menú ☰ → Configuración y actividad
  2. Aplicaciones y sitios web → buscá cualquier autorización otorgada
  3. Remové el acceso o simplemente evitá comentar la palabra clave

Si adicionalmente querés confirmar por escrito la eliminación de cualquier registro operativo de logs, escribí a carloshuanesarmiento@gmail.com con el asunto data-deletion y tu username de Instagram. Se responde en máximo 7 días hábiles.
